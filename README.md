# omf-mes — MES 시장조사 리포트

한국·글로벌 주요 MES(제조실행시스템) 제품의 핵심 기능 모듈과 **AI Agent 적용 범위**를
다중 출처로 조사·검증한 리포트. CREFLE Presentation Design System(브랜드 레드 `#C9252C`,
Spoqa Han Sans Neo, Material Design 3 토큰) 기반 정적 웹페이지.

## 보기

- **GitHub Pages:** <https://crefleinc.github.io/omf-mes/> *(Pages 활성화 후)*
- **로컬:** `index.html`을 브라우저로 열거나 `python3 -m http.server` 후 접속

## 구성

| 경로 | 설명 |
| --- | --- |
| `index.html` | 리포트 본문 (스크롤형 단일 페이지) |
| `assets/css/crefle.css` | CREFLE 디자인 토큰 (Spoqa 웹폰트 CDN) |
| `assets/*.svg` | CREFLE 로고 마크·워드마크 |
| `.nojekyll` | GitHub Pages Jekyll 처리 비활성화 |

## 방법

5개 검색 각도 → 26개 출처 fetch → 109개 주장 추출 → 25개 3표 적대적 검증
(20개 확정, 5개 반박). 자세한 주의사항·반박 주장은 리포트 본문 §06 참조.
